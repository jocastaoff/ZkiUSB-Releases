4. AVERTISSEMENTS IMPORTANTS
--------------------------------------------------------------------------------

⚠ OPÉRATIONS À RISQUE

ZkiUSB effectue des opérations sur les périphériques de stockage qui 
peuvent entraîner une PERTE IRRÉVERSIBLE DE DONNÉES :

    • FORMATAGE : Efface définitivement toutes les données
    • RECONSTRUCTION DE PARTITIONS : Peut corrompre la table des partitions
    • NETTOYAGE FORCÉ : Supprime irrémédiablement toutes les partitions
    • RÉPARATION DU SYSTÈME DE FICHIERS : Risque de perte de données

⚠ RECOMMANDATIONS OBLIGATOIRES

AVANT toute opération avec ZkiUSB, vous DEVEZ :

    1. Sauvegarder l'intégralité de vos données importantes
    2. Vérifier l'intégrité de vos sauvegardes
    3. Vous assurer de disposer des droits d'accès sur le périphérique
    4. Comprendre l'opération que vous allez effectuer
